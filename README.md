
Project Title -
Handwritten Digit Recognition using CNN

Project Description : 
This project builds a Convolutional Neural Network (CNN) to recognize handwritten digits from images using the MNIST dataset.
The project also includes Exploratory Data Analysis (EDA) and a Streamlit web application that allows users to upload handwritten digit images and receive real-time predictions.

Features
-CNN model for handwritten digit classification
-Dataset exploration and visualization using EDA
-Image preprocessing pipeline (grayscale conversion, normalization, resizing)
-Interactive web interface using Streamlit
-Real-time prediction of uploaded digit images

Technologies Used
-Python
-TensorFlow / Keras
-NumPy
-Pandas
-Matplotlib
-Seaborn
-Streamlit

Workflow
-Perform EDA to understand digit distribution.
-Train a CNN model on MNIST dataset.
-Preprocess uploaded images to match MNIST format.

Predict digit using trained model.

Display prediction through Streamlit interface.
